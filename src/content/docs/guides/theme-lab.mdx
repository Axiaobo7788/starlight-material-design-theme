Theme Lab is a dense page for visual checks. It keeps Starlight's default
navigation, table of contents, markdown, and component markup active while the
theme applies Material Design 3 color roles, tonal surfaces, shape, state layers,
and compact documentation density.

## Asides

<Aside type="note" title="Neutral surface">
	Notes use a neutral container to carry supporting context. The border is an
	inset tonal outline rather than a heavy stripe.
</Aside>

<Aside type="tip" title="Primary container">
	Tips use the primary container family. They should feel helpful and selected,
	not promotional.
</Aside>

<Aside type="caution" title="Secondary warning">
	Caution blocks use a warm secondary role so they can stand apart without
	copying the error palette.
</Aside>

<Aside type="danger" title="Error role">
	Danger blocks use error container roles and must keep readable foreground
	contrast in both themes.
</Aside>

## Tables

| Surface | Expected role | Regression risk |
| --- | --- | --- |
| Header | `surface-container-low` | Loading-time height shift |
| Sidebar selected item | `primary-container` | Flashing between secondary and primary roles |
| TOC active item | `primary` | Last item fails to activate near page bottom |
| Table header | `surface-container-high` | Header blends into table body |
| Code block | `surface-container-low` | Syntax colors lose contrast |

## Code Blocks

Inline `plugins: [md3Theme()]` remains the intended user API. Users should not
need to configure Tailwind to use the theme.

```ts
import starlight from '@astrojs/starlight';
import md3Theme from 'starlight-theme-md3';

export default {
	integrations: [
		starlight({
			plugins: [
				md3Theme({
					seed: '#00a99d',
					variant: 'tonalSpot',
					density: 'compact',
				}),
			],
		}),
	],
};
```

## Chinese Paragraphs

这一段用于观察中文正文在紧凑文档布局中的表现。标题不应该过大，正文不应该显得松散，列表、表格和代码块之间也需要有清楚的层次。对于长期维护的技术文档来说，稳定的阅读节奏比一眼惊艳更重要。

中文链接例如 [Design Tokens](../../reference/design-tokens/) 应该与英文链接拥有一致的颜色和下划线策略。`--md-sys-color-primary` 这类 token 出现在中文句子里时，行高和背景不应该产生跳动。
