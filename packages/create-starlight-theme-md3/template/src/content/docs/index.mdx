## Theme Surface

<CardGrid>
	<Card title="Starlight native" icon="puzzle">
		Navigation, search, table of contents, and MDX components remain Starlight-native.
	</Card>
	<Card title="Material roles" icon="code-branch">
		Map Material Design 3 `--md-sys-*` roles into Starlight's existing custom properties.
	</Card>
	<Card title="Readable density" icon="list-format">
		Keep docs compact enough for reference pages while preserving generous hit targets.
	</Card>
	<Card title="No heavy runtime" icon="server">
		Use Material Web as a design reference without shipping it as a required component runtime.
	</Card>
</CardGrid>
